##  Interactive 3D Universe Visualization

Explore the universe from quantum particles to cosmic scales in real-time 3D!

### Features
- **Scale Navigation**: Zoom from Planck length (10⁻³⁵m) to observable universe (10²⁷m)
- **Real Physics**: Accurate planetary orbits and relative sizes
- **Educational**: Scientific data for each object (mass, density, temperature)
- **Interactive**: Mouse drag to rotate, scroll to zoom, keyboard shortcuts
- **Responsive**: Works on desktop and mobile devices

### Technologies
- **Three.js** - WebGL 3D rendering
- **JavaScript (ES6+)** - Physics and interactivity
- **HTML5 Canvas** - Graphics rendering
- **CSS3** - Modern UI with animations

### Controls
| Action | Control |
|--------|---------|
| Rotate View | Click + Drag mouse |
| Zoom In/Out | Scroll wheel |
| Time Speed Down | Q key |
| Time Speed Up | E key |
| Toggle Grid | G key |
| Toggle Orbits | O key |
| Reset View | R key |
| Auto-rotation | Spacebar |
